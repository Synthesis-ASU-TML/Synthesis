O4.rhyth_abs - Feb 2014

DEPENDENCIES:
CNMAT externals - http://cnmat.berkeley.edu/downloads
FTM Library - download at http://ftm.ircam.fr/index.php/Download
createfolder - included (JASCH)

INSTRUCTIONS:
1. Download and Install latest version of Max/MSP (run in 32bit mode) + odot externals included in CNMAT library (http://cnmat.berkeley.edu/downloads) + FTM Library (http://ftm.ircam.fr/index.php/Download)
2. make sure O4.rhyth_abs folder + dependencies are in your max search path

INCLUDED:
rhythRecord - record and store a rhythmic stream (time series or 1D data)
rhythPhrase - parse recorded data by selection, phrase, or recalling previously saved data
rhythPlay - playback selected data
rhythAnalysis - analysis of playback stream
